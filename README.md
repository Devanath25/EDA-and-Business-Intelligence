# 📊 ApexPlanet Data Analytics Internship - Task 2
## Exploratory Data Analysis (EDA) & Business Intelligence

### 📌 Project Overview

This repository contains the work completed for **Task 2** of the **ApexPlanet Data Analytics Internship Program**.

The objective of this task is to perform **Exploratory Data Analysis (EDA)** on the cleaned dataset, answer business questions using SQL, discover meaningful insights through visualization, and propose a dashboard that supports business decision-making.

---

## 🎯 Objectives

- Perform descriptive statistical analysis
- Explore numerical and categorical variables
- Identify trends and patterns through visualization
- Answer business questions using SQL
- Analyze relationships between variables
- Design a business dashboard mock-up

---
